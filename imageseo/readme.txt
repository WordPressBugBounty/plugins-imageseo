== Frequently Asked Questions ==

= What is an alternative text (alt) ? =
An alt tag, also known as "alt attribute" and "alt description," is an HTML attribute applied to image tags to provide a text alternative for search engines. Images with alt texts positively impact SEO. (ie : <img alt="This is the logo of ImageSEO, the best image SEO plugin for WordPress">).

= Do I lose my optimizations if I downgrade my plan ? =

Of course not !

= What is the difference between alt text and alt tag ? =

Alt text (alternative text), also known as "alt attribute," “alt description", or technically incorrectly as "alt tag”, is used within a HTML code to describe the appearance and function of an image on a page. There is no difference between alt text and alt tag.

= How to write alt text ? =

Alt text should be short but descriptive. You need to describe your image in 2-3 words.
